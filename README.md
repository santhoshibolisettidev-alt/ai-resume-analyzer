# 🚀 AI Resume Analyzer

An intelligent full-stack web application that allows users to upload resumes and receive automated analysis, skill extraction, and performance scoring. This project is designed to help job seekers evaluate their resumes and improve their chances of getting shortlisted by recruiters.

---

## 📌 Project Overview

The **AI Resume Analyzer** is a modern web application built using the MERN-style architecture (React + Node.js + MySQL). It enables users to securely upload their resumes in PDF format, analyze content, and visualize key insights such as skills, word count, and overall resume score.

The system simulates real-world resume screening by extracting relevant data and presenting it in a user-friendly dashboard.

## ✨ Features

### 🔐 Authentication System
- User Registration and Login  
- Secure authentication using JWT tokens  
- Protected routes for authorized access  

### 📄 Resume Upload
- Upload resumes in PDF format  
- File handling using Multer  
- Stores resume metadata in database  

### 📊 Dashboard Analytics
- Total resumes uploaded  
- Resume score calculation  
- Skills extracted from resume  
- Word count and performance indicators  

### 📜 Resume History
- View all uploaded resumes

- Fully responsive design  
- Clean and professional interface

- Smooth user experience  

---

## 🔄 Application Flow

1. User registers and logs in  
2. Uploads a resume (PDF)  
3. Backend processes the file  
4. Extracts key data (skills, text, etc.)  
5. Stores data in MySQL database  
6. Displays analysis on dashboard
